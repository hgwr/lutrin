Lutrin Help

"Lutrin" is a simple image viewer. You can view images sequentially in a folder by using left/right arrow keys.

In addition, you can directly open a zip file. Archiving image files to a zip file, you can view images sequentially in the zip file.

Menu Menu Item Key Function
Lutrin About Lutrin Show dialog about Lutrin.
Quit Lutrin ⌘Q Quit application.
File Open... ⌘O Open a image or zip file.
Close ⌘W Quit application.
View Single S Switch to a single image view.
Left to Right L Switch to a spread display. Pages go from left to right.
Right to Left R Switch to a spread display. Pages go from right to left.
Actual Size 0 Zooms the image so that it is displayed using its true size.
Fit to Window 9 Zooms the image so that it fits in the image view.
Zoom In ↑ Zooms the image in.
Zoom Out ↓ Zooms the image out.
Next → Display the next image in the folder.
Previous ← Display the previous image in the folder.
First ↖ (home key) Display the first image in the folder.
Last ↘ (end key) Display the last image in the folder.
Full-screen F Switch between windowed and full-screen display.
