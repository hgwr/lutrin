Lutrin ヘルプ

Lutrin は画像ビューアーです。矢印キーを使ってフォルダ内にある画像を順に閲覧することができます。

見開き表示モードにすれば、ウィンドウの中に２つの画像を表示することができ、本を開いて読む感覚で画像を閲覧することができます。

また、 zip ファイルを直接開くことができます。画像ファイルを zip ファイルにアーカイブしておくと、 通常のフォルダ内にある画像を閲覧するのと同じように zip ファイル中の画像を順に閲覧することができます。

メニュー メニュー項目 キー 機能
Lutrin About Lutrin Lutrin についてのダイアログを表示します。
Quit Lutrin ⌘Q Lutrin を終了します。
File Open... ⌘O 画像ファイルまたは zip ファイルを開きます。
Close ⌘W Lutrin を終了します。
View Single S 単一画像表示にします。
Left to Right L 左から右へ進む見開き表示にします。
Right to Left R 右から左へ進む見開き表示にします。
Actual Size 0 画像を実際のサイズで表示します。
Fit to Window 9 画像をウィンドウサイズに合わせて表示します。
Zoom In ↑ 画像を拡大します。
Zoom Out ↓ 画像を縮小します。
Next → フォルダ内にある次の画像を表示します。
Previous ← フォルダ内にある前の画像を表示します。
First ↖ (home キー) フォルダ内にある最初の画像を表示します。
Last ↘ (end キー) フォルダ内にある最後の画像を表示します。
Full-screen F フルスクリーン表示とウィンドウ表示を切り替えます。
